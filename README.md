# HUL Sales Performance Dashboard

## Project Overview

This Tableau dashboard provides an end-to-end view of sales performance, target achievement, product growth, and business KPIs. The solution enables stakeholders to monitor sales trends, identify top-performing products, and take corrective actions on underperforming categories.

## Business Problem

Business teams required a centralized dashboard to:

- Monitor YTD sales performance
- Track target achievement
- Analyze product-level growth
- Identify top and bottom-performing products
- Enable data-driven decision making

## Dashboard Features

### Executive KPIs

- YTD Sales
- YTD Quantity
- Target Achievement %
- Year-over-Year Comparison

### Dynamic KPI Switching

Users can dynamically switch between:

- Sales
- Quantity
- Growth %
- Target Achievement %

### Product Performance Analysis

- Top 5 Performing Products
- Underperforming Products
- Product-wise Growth Analysis

### Interactive Filters

- Date Selection
- Time Period Selection
- Dynamic KPI Selection

## Tools & Technologies

- Tableau Desktop
- Tableau Prep
- SQL
- Excel
- Data Modeling
- Data Visualization

## Key Insights

- Identified top-performing products contributing significantly to overall revenue.
- Highlighted underperforming products requiring corrective action.
- Improved business visibility through centralized KPI tracking.
- Enabled faster decision-making through interactive visual analytics.
